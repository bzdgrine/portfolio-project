1. CSS Comments

Understanding CSS Comments:

/* Comments are used to explain the code, and may help when you edit the source code at a later date */
/* Comments are ignored by browsers */
/* A CSS comment starts with /* and ends with */. Comments can also span multiple lines */

p {
    color: red;
    /* This is a single-line comment */
    text-align: center;
}

/* This is
   a multi-line
   comment */

This text demonstrates the CSS properties with comments

2. Three Ways to Insert CSS

1. External Style Sheet:

<!-- In HTML file -->
<head>
    <link rel="stylesheet" type="text/css" href="styles.css">
</head>

/* In styles.css file */
body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
}

2. Internal Style Sheet:

<head>
    <style type="text/css">
        body {
            background-color: #f0f0f0;
            font-family: Arial, sans-serif;
        }
    </style>
</head>

3. Inline Style:

<p style="color: red; text-align: center;">This is a paragraph with inline styles</p>

Examples of each method:

1. External CSS (linked in head section)

2. Internal CSS (style tag in head section)

3. Inline CSS (style attribute)

7. CSS Grid Layout

Advanced Grid Layouts:

.grid-container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
    grid-gap: 20px;
    grid-auto-rows: minmax(100px, auto);
}

/* Named Grid Areas */
.container {
    display: grid;
    grid-template-areas: 
        "header header header"
        "nav main aside"
        "footer footer footer";
    grid-template-columns: 200px 1fr 200px;
    grid-template-rows: auto 1fr auto;
    min-height: 100vh;
}

.header { grid-area: header; }
.nav { grid-area: nav; }
.main { grid-area: main; }
.aside { grid-area: aside; }
.footer { grid-area: footer; }

8. CSS Animations

Keyframe Animations:

@keyframes slide-in {
    0% {
        transform: translateX(-100%);
        opacity: 0;
    }
    100% {
        transform: translateX(0);
        opacity: 1;
    }
}

.animated-element {
    animation: slide-in 0.5s ease-out forwards;
}

/* Multiple Animations */
.multi-animated {
    animation: 
        slide-in 0.5s ease-out,
        fade-out 0.3s ease-in 2s forwards;
}

/* Animation Properties */
.custom-animation {
    animation-name: slide-in;
    animation-duration: 0.5s;
    animation-timing-function: ease-out;
    animation-delay: 0.2s;
    animation-iteration-count: infinite;
    animation-direction: alternate;
    animation-fill-mode: forwards;
}
Animated Element

9. CSS Custom Properties (Variables)

Using CSS Variables:

:root {
    --primary-color: #ff4444;
    --secondary-color: #333;
    --spacing-unit: 8px;
    --border-radius: 4px;
    --max-width: 1200px;
}

.component {
    color: var(--primary-color);
    margin: calc(var(--spacing-unit) * 2);
    border-radius: var(--border-radius);
}

/* Fallback Values */
.fallback-example {
    color: var(--undefined-color, #666);
}

/* Changing Variables with Media Queries */
@media (max-width: 768px) {
    :root {
        --spacing-unit: 4px;
    }
}

10. CSS Transforms and Transitions

3D Transforms and Transitions:

/* 3D Transforms */
.transform-3d {
    transform: 
        perspective(1000px)
        rotateX(45deg)
        rotateY(45deg)
        translateZ(50px);
}

/* Multiple Transforms */
.multi-transform {
    transform:
        scale(1.2)
        rotate(45deg)
        translate(100px, 200px);
}

/* Complex Transitions */
.complex-transition {
    transition:
        transform 0.3s ease-out,
        background-color 0.5s linear,
        box-shadow 0.3s ease-in;
}

.complex-transition:hover {
    transform: scale(1.1) rotate(5deg);
    background-color: var(--primary-color);
    box-shadow: 0 10px 20px rgba(0,0,0,0.2);
}
