HTML Styles and Tables

Understanding HTML Styling and Table Structures

1. HTML Styles

Basic Syntax:

<tagname style="property:value;">

The HTML style attribute is used to add styles to an element, such as color, font, size, and more.

Background Color

Code:

<h1 style="background-color:cyan;">This is a heading</h1>
<p style="background-color:tomato;">This is a paragraph.</p>

This is a heading

This is a paragraph.

Text Color and Fonts

Code:

<p style="color:blue; font-family:verdana;">Styled text</p>

Styled text

2. HTML Tables

Basic Table Structure:

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>
Header 1 Header 2
Data 1 Data 2

Table Styling

CSS for Tables:

table, th, td {
    border: 1px solid black;
    border-collapse: collapse;
    padding: 15px;
}

Spanning Columns

Code:

<th colspan="2">Spanning Header</th>
Column 1 Spanning Header
Data 1 Data 2 Data 3

3. Advanced Table Features

Table with Caption

Table with Caption Example:

<table>
    <caption>Monthly Savings</caption>
    <tr>
        <th>Month</th>
        <th>Savings</th>
    </tr>
    <tr>
        <td>January</td>
        <td>$100</td>
    </tr>
</table>
Monthly Savings
Month Savings
January $100

Table with Header, Body, and Footer

Structured Table Example:

<table>
    <thead>
        <tr>
            <th>Item</th>
            <th>Price</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Coffee</td>
            <td>$3.50</td>
        </tr>
        <tr>
            <td>Tea</td>
            <td>$2.50</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td>Total</td>
            <td>$6.00</td>
        </tr>
    </tfoot>
</table>
Item Price
Coffee $3.50
Tea $2.50
Total $6.00

Responsive Tables

Making Tables Responsive:

.table-responsive {
    overflow-x: auto;
    max-width: 100%;
}

/* For small screens */
@media screen and (max-width: 600px) {
    table {
        display: block;
    }
    thead {
        display: none;
    }
    tr {
        display: block;
        margin-bottom: 15px;
    }
    td {
        display: block;
        text-align: right;
        padding-left: 50%;
        position: relative;
    }
    td::before {
        content: attr(data-label);
        position: absolute;
        left: 0;
        width: 50%;
        text-align: left;
    }
}

Table with Row and Column Spans

Complex Table Structure:

<table>
    <tr>
        <th colspan="2">Name</th>
        <th rowspan="2">Age</th>
    </tr>
    <tr>
        <th>First</th>
        <th>Last</th>
    </tr>
    <tr>
        <td>John</td>
        <td>Doe</td>
        <td>30</td>
    </tr>
</table>
Name Age
First Last
John Doe 30
