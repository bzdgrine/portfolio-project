<html lang="en">

    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header>
        <nav>Navigation</nav>
    </header>
    
    <main>
        <article>
            <section>Content Section</section>
        </article>
        <aside>Sidebar</aside>
    </main>
    
    <footer>Footer Content</footer>
</body>
</html>

9. Display Italicized Text

Six different elements of displaying italicized text with distinct purpose:

<i>The italics element is a physical element for displaying italicized text.</i>

<em>The emphasis element is a logical element for emphasizing important portions of a document.</em>

<cite>The citation element identifies a portion of your document as a reference to an outside source.</cite>

<var>The logical element indicates a variable, as might be used in computer code.</var>

<dfn>This logical element identifies a portion of text as defining instance of a term.</dfn>

<address>To set apart your address or personal information at the bottom of a Web page. This element also generally adds a line break before and after the address.</address>
