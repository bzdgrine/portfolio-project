1. HTML Lists

Ordered Lists

Basic Ordered List:

<ol>
    <li>First Item</li>
    <li>Second Item</li>
    <li>Third Item</li>
</ol>
  1. First Item
  2. Second Item
  3. Third Item

Ordered List with Different Types:

<ol type="A" start="3">
    <li>Item C</li>
    <li>Item D</li>
</ol>

<ol type="i">
    <li>Roman Item 1</li>
    <li>Roman Item 2</li>
</ol>
  1. Item C
  2. Item D
  1. Roman Item 1
  2. Roman Item 2

Unordered Lists

Different List Styles:

<ul style="list-style-type: disc">
    <li>Disc Bullet</li>
</ul>
<ul style="list-style-type: circle">
    <li>Circle Bullet</li>
</ul>
<ul style="list-style-type: square">
    <li>Square Bullet</li>
</ul>
  • Disc Bullet
  • Circle Bullet
  • Square Bullet

Description List Example:

    <dd>HyperText Markup Language - the standard markup language for web pages</dd>

<!-- Absolute URL -->

<!-- Relative URL -->

3. HTML Images

Basic Image Syntax

Basic Image Tag:

<img src="image.jpg" alt="Description of image">

Image with Size

Image with Width and Height:

<img src="image.jpg" alt="Sized image" width="200" height="100">

Image as Link

Image Inside Link:

<a href="https://example.com">
    <img src="image.jpg" alt="Clickable image">
</a>

Image Formats

Common Image Formats:

WebP: Best for web, supports both lossy and lossless compression
JPEG: Best for photographs and complex images
PNG: Best for images with transparency
GIF: Best for simple animations and images with few colors

4. Advanced List Styling

Custom List Styling:

/* Custom Bullet Points */
ul.custom-bullets {
    list-style: none;
}

ul.custom-bullets li::before {
    content: "➜";
    color: red;
    font-weight: bold;
    margin-right: 10px;
}

/* Custom Numbering */
ol.custom-numbers {
    list-style: none;
    counter-reset: my-counter;
}

ol.custom-numbers li::before {
    counter-increment: my-counter;
    content: "Step " counter(my-counter) ": ";
    color: red;
    font-weight: bold;
}

5. Advanced Link Features

Link Attributes and States:

<!-- Download Links -->
<a href="file.pdf" download="custom-name.pdf">Download PDF</a>

<!-- Link to Specific Section -->
<a href="#section-id">Jump to Section</a>

<!-- Link with Language Attribute -->
<a href="spanish.html" hreflang="es">Versión Española</a>

<!-- Link with Relationship -->
<a href="terms.html" rel="nofollow">Terms of Service</a>

/* Link States */
a:link { color: blue; }
a:visited { color: purple; }
a:hover { color: red; }
a:active { color: green; }

6. Advanced Image Techniques

Responsive Images and Art Direction:

<!-- Responsive Images -->
<img srcset="small.jpg 300w,
             medium.jpg 600w,
             large.jpg 900w"
     sizes="(max-width: 500px) 300px,
            (max-width: 900px) 600px,
            900px"
     src="fallback.jpg"
     alt="Responsive image">

<!-- Picture Element for Art Direction -->
<picture>
    <source media="(min-width: 800px)" srcset="hero.jpg">
    <source media="(min-width: 400px)" srcset="medium.jpg">
    <img src="mobile.jpg" alt="Art directed image">
</picture>

<!-- Lazy Loading -->
<img src="image.jpg" loading="lazy" alt="Lazy loaded image">

/* Image Filters */
.image-filters {
    filter: brightness(110%) contrast(110%) saturate(120%);
}

/* Image Hover Effects */
.image-hover {
    transition: transform 0.3s ease;
}
.image-hover:hover {
    transform: scale(1.1);
}
