HTML Forms and Multimedia

Interactive Elements and Rich Media Content

1. HTML Forms

Web Forms are interactive elements that allow users to input data and submit it for processing.

Form Fundamentals

Basic Form Structure:

<form action="/action_page.php" method="post">
    <label for="username">Username:</label>
    <input type="text" id="username" name="username">
    <input type="submit" value="Submit">
</form>

Form Controls

Different Input Types:

<form>
    <!-- Text Input -->
    <input type="text" placeholder="Enter text">

    <!-- Password Input -->
    <input type="password" placeholder="Enter password">

    <!-- Radio Buttons -->
    <input type="radio" name="gender" value="male"> Male
    <input type="radio" name="gender" value="female"> Female

    <!-- Checkboxes -->
    <input type="checkbox" name="vehicle" value="bike"> I have a bike
    <input type="checkbox" name="vehicle" value="car"> I have a car
</form>

Form Attributes

Common Form Attributes:

<form action="/submit.php" method="post" target="_blank" autocomplete="on">
    <input type="text" name="username" required maxlength="30" size="20">
    <input type="submit" value="Submit">
</form>

3. Advanced Form Controls

HTML5 Form Input Types:

<form>
    <!-- Date and Time Inputs -->
    <input type="date">
    <input type="time">
    <input type="datetime-local">
    <input type="month">
    <input type="week">

    <!-- Number Inputs -->
    <input type="number" min="0" max="100" step="5">
    <input type="range" min="0" max="100">

    <!-- Color Picker -->
    <input type="color">

    <!-- File Upload -->
    <input type="file" accept="image/*" multiple>

    <!-- Search Input -->
    <input type="search" placeholder="Search...">
</form>

4. Form Validation

HTML5 Form Validation:

<form>
    <!-- Required Fields -->
    <input type="text" required>

    <!-- Pattern Matching -->
    <input type="text" pattern="[A-Za-z]{3}">

    <!-- Email Validation -->
    <input type="email" multiple>

    <!-- Length Constraints -->
    <input type="text" minlength="3" maxlength="8">

    <!-- Custom Validation Message -->
    <input type="text" pattern="[0-9]+" title="Numbers only">
</form>

/* CSS for Valid/Invalid States */
input:valid {
    border-color: green;
}

input:invalid {
    border-color: red;
}

5. Advanced Multimedia Features

Video and Audio Controls:

<!-- Video with Advanced Controls -->
<video controls crossorigin playsinline poster="thumbnail.jpg">
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    <track kind="subtitles" src="captions.vtt" srclang="en" label="English">
</video>

<!-- Audio with Advanced Controls -->
<audio controls preload="metadata">
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
</audio>

/* Custom Media Controls */
.custom-video-player {
    width: 100%;
    max-width: 800px;
    position: relative;
}

.custom-controls {
    position: absolute;
    bottom: 0;
    width: 100%;
    background: rgba(0,0,0,0.5);
}

6. Responsive Multimedia

Making Media Responsive:

/* Responsive Video Container */
.video-container {
    position: relative;
    padding-bottom: 56.25%; /* 16:9 Aspect Ratio */
    height: 0;
    overflow: hidden;
}

.video-container iframe,
.video-container video {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
}

/* Picture Element for Art Direction */
<picture>
    <source media="(min-width: 800px)" srcset="large.jpg">
    <source media="(min-width: 400px)" srcset="medium.jpg">
    <img src="small.jpg" alt="Responsive image">
</picture>
