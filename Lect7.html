Practical Use of Bootstrap Layout, Grid, and Colors

1. Bootstrap Containers in Practice

Bootstrap containers are used to pad and align your content. There are two main types: .container (fixed width) and .container-fluid (full width).

Fixed Container Example:

<div class="container">
    <h1>Fixed Container</h1>
    <p>This is a fixed-width container.</p>
</div>

Fixed Container

This is a fixed-width container.

Fluid Container Example:

<div class="container-fluid">
    <h1>Fluid Container</h1>
    <p>This is a full-width container.</p>
</div>

Fluid Container

This is a full-width container.

Container with Padding, Border, and Color:

<div class="container p-3 my-3 border bg-primary text-white">
    <h1>Styled Container</h1>
    <p>This container uses Bootstrap spacing, border, and color classes.</p>
</div>

Styled Container

This container uses Bootstrap spacing, border, and color classes.

2. Bootstrap Grid System in Practice

The Bootstrap grid system uses rows and columns to layout and align content. It's built with flexbox and allows up to 12 columns across the page.

Three Equal Columns:

<div class="row">
    <div class="col">Column 1</div>
    <div class="col">Column 2</div>
    <div class="col">Column 3</div>
</div>
Column 1
Column 2
Column 3

Two Unequal Responsive Columns:

<div class="row">
    <div class="col-sm-4">col-sm-4</div>
    <div class="col-sm-8">col-sm-8</div>
</div>
col-sm-4
col-sm-8

Responsive Four Columns (Stack on Mobile):

<div class="row">
    <div class="col-sm-3">col-sm-3</div>
    <div class="col-sm-3">col-sm-3</div>
    <div class="col-sm-3">col-sm-3</div>
